## Dependencies

- axios v1.6.8
- sonner v1.4.41

# Quick start

1. Clone the repo:

```shell
git clone https://github.com/danielortega30/reservamos-test.git
```

2. Open the project

```shell
cd reservamos-test
```

3. Install dependencies

```shell
npm i
```

4. Run project

```shell
npm run dev
```
